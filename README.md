# Vaccination Data Analysis

## Overview

This project analyzes vaccination data to identify trends, compare vaccination progress across regions, and generate meaningful insights using Python.

## Objectives

- Clean and preprocess vaccination data
- Perform exploratory data analysis (EDA)
- Visualize vaccination trends
- Compare vaccination statistics across countries/states
- Generate data-driven insights

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Features

- Data Cleaning
- Missing Value Handling
- Trend Analysis
- Data Visualization
- Statistical Summary
